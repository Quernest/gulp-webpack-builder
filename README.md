## Front-End workspace

### Installation with yarn
<code>yarn install</code>

### or npm
<code>npm install</code>

### Contains
* Gulp
* Webpack
* Babel
* SCSS
* Pug

### Commands
* <code> yarn dev </code>
* <code> yarn build </code>
* <code> yarn clean </code>

### Source folder structure
* src
  * assets
    * images
    * fonts
  * js
    * *.js
  * styles
    * *.scss
  * views
    * *.pug
