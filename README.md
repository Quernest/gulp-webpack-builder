### Front-End workspace
<hr>

#### Installation with yarn
<code>yarn install</code>

#### or npm
<code>npm install</code>

### Contains
* Gulp 4.x.x
* Webpack-Stream 4.x.x
* Babel 6.x.x
* SCSS preprocessor
* Pug

### Commands
* <code> yarn dev </code> - development
* <code> yarn build </code> - production
* <code> yarn clean </code> - delete dist folder
